### Запуск
1. `git clone https://github.com/GeorgiySurkov/photo_album.git <dir_name>`
2. `cd <dir_name>`
3. `python -m venv venv`
4. `venv\scripts\activate`
5. `pip install -r requirements.txt`
6. `set FLASK_APP=main.py`
7. `flask run`
